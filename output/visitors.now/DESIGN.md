# Design System

## Overview
Design tokens extracted from visitors.now.

## Colors
- **Primary** (#ffa600): CTAs, active states, key interactive elements
- **Secondary** (#2c78fc): Supporting UI, secondary actions
- **Surface** (#ffffff): Page backgrounds
- **On-surface** (#181925): Primary text

## Typography
- **Headlines**: OpenRunde, semi-bold, 12–60px

## Components
- **Buttons**: Fully rounded, primary uses #9580ff fill, outlined variant available
- **Border radius scale**: 4px, 6px, 8px, 12px
- **Elevation**: uses box shadows for depth

## Do's and Don'ts
- Do use rounded corners consistently across interactive elements
- Do use the primary color sparingly — only for the most important action per screen
- Do maintain 4.5:1 contrast ratio for all body text (WCAG AA)
