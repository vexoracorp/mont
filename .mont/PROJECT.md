# Mont — Project Context

> 이 문서는 AI 에이전트가 프로젝트 맥락을 빠르게 파악하기 위한 참조 문서입니다.
> 세션이 바뀌어도 이 파일을 읽으면 프로젝트 전체 맥락을 복원할 수 있습니다.

---

## 제품 개요

**Mont**는 모든 이커머스 디지털 프로덕트의 자동배송을 관리하는 올인원 SaaS 플랫폼이다.

네이버 스토어, G2G, G2A, 개인 웹사이트 등 어디서 팔든 — 주문을 실시간으로 추적하고, 디지털 상품을 자동으로 배송하고, 모든 채널을 한 곳에서 관리할 수 있다.

## 핵심 가치

- **자동배송**: 주문 들어오면 디지털 상품(키, 계정, 라이선스, 기프트카드 등)을 즉시 자동 전달
- **멀티 플랫폼 통합**: 네이버, G2G, G2A, 개인사이트 등 모든 판매 채널을 하나로
- **효율성**: 수동 작업 제거, 24/7 무인 운영 가능

## 취급 상품 (디지털 프로덕트)

- 게임 키 / 계정 (Steam, PlayStation, Xbox 등)
- 소프트웨어 라이선스 (Windows, Office, Adobe 등)
- 기프트카드 / 포인트 (구글플레이, 아이튠즈, 문화상품권 등)
- 인게임 아이템 / 골드
- 디지털 콘텐츠 전반 — 종류 불문

## 타겟 유저

| 세그먼트 | 설명 |
|---------|------|
| 개인 셀러 | 네이버 스토어, 개인 사이트에서 소규모로 파는 사람 |
| 전문 리셀러 | G2G, G2A 같은 마켓플레이스에서 대량 판매하는 사람 |
| 기업/팀 | 디지털 상품 유통 회사, 팀 단위 운영 |

## 킬러 기능

1. **자동배송** — 주문 감지 → 재고에서 상품 꺼내기 → 구매자에게 즉시 전달
2. **멀티 플랫폼 통합** — 네이버, G2G, G2A, 개인사이트 등 한 곳에서 관리
3. **재고 관리** — 디지털 상품 재고 자동 추적 + 품절 알림
4. **주문 추적** — 모든 플랫폼 주문을 실시간 통합 대시보드
5. **매출 분석** — 플랫폼별 매출, 수익률, 트렌드 분석
6. **API 연동** — 외부 시스템/봇과 연동 가능한 API 제공
7. **가격 자동 조정** — 경쟁가 모니터링 + 자동 가격 조정

## 브랜드

- **톤**: 프로페셔널 + 빠르고 효율적. 신뢰감 있으면서 테크 느낌.
- **포지셔닝**: "디지털 상품 판매의 모든 것을 자동화"
- **경쟁사**: 직접 경쟁사 없음/미파악 (블루오션 가능성)
- **단계**: MVP 개발 중

## 기술 스택

- Runtime: Bun
- Framework: Vite + React 19
- Language: TypeScript 6
- Styling: Tailwind CSS v4
- Components: shadcn/ui (base-nova style)
- Font: Plus Jakarta Sans Variable
- Icons: Lucide React
- Path alias: `@/*` → `./src/*`
